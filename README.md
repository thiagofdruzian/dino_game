# Jogo do Dino (Pygame)

Este projeto é uma recriação simples do famoso "jogo do dinossauro" do Google Chrome — desenvolvido **apenas para fins de estudo** da linguagem Python e da biblioteca Pygame.

## Objetivo do Projeto

O objetivo principal deste projeto é **praticar conceitos fundamentais de programação** como:

- Lógica de jogos 2D
- Manipulação de imagens e sprites
- Eventos de teclado
- Animações simples
- Detecção de colisão (em versões futuras)
- Uso do módulo `pygame`
- Estruturação de código em Python
- Versionamento com Git e GitHub

---

## Tecnologias utilizadas

- Python 3.13+
- [Pygame](https://www.pygame.org/) (biblioteca para jogos 2D)
- Git e GitHub (controle de versão)

---

## Como executar o projeto?

### 1. Pré-requisitos

- Python instalado ([Download Python](https://www.python.org/downloads/))
- Pygame instalado (você pode instalar com o comando abaixo):

```bash
